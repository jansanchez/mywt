# mywt
my first webtask

Set a version to the `url` attribute of css files. Essential for when working with CDN.

# Install

```
npm install -d
```

# How to use

## Running the express server

```
npm start
```

## Webpage

```
http://localhost:3000/
```


![how to use](docs/img/wt.gif)
